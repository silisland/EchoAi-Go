# RV1106边缘龙虾助手 - 技术栈（简化版）

## 一、软件技术栈

### 核心系统
- **操作系统**：嵌入式Linux（Buildroot定制）
- **开发语言**：C/C++（底层）、Python（AI服务）、Go（Agent框架）
- **构建工具**：CMake、Git

### AI能力
- **语音识别**：开源ASR模型（本地）+ 云端API备份
- **语音合成**：云端TTS服务
- **大模型对话**：轻量化Agent框架 + 云端LLM
- **物体检测**：YOLOv5算法 + NPU加速

### 应用框架
- **GUI界面**：LVGL图形框架
- **通信协议**：WebSocket + JSON
- **音频处理**：Opus编解码 + PortAudio采集

## 三、关键技术组合

```
【输入层】
麦克风 → 摄像头 → 触摸屏

【处理层】
语音识别(ASR) → AI对话(Agent) → 物体检测(YOLOv5)

【输出层】
扬声器 ← GUI界面 ← LVGL

【核心】
云端API <——> WebSocket <——> 本地AI框架
```
